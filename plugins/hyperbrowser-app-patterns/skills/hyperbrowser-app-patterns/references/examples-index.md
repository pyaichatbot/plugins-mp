# Hyperbrowser Examples Index

Curated index from `hyperbrowserai/hyperbrowser-app-examples`.

## Browser Automation And Scraping

- `deep-crawler-bot` - Crawl a site and show progress/results.
- `site-to-dataset` - Crawl, scrape, chunk, and export website content.
- `scrape-to-api` - Generate API-like access from scraped site content.
- `assets-optimizer` - Analyze and optimize discovered site assets.
- `competitor-tracker` - Monitor competitor pages, diff HTML, summarize changes, notify.

## Research And Intelligence

- `deep-reddit-researcher` - Research Reddit conversations with screenshots/progress.
- `deep-job-researcher` - Match resume/candidate data against live job listings.
- `yc-research-bot` - Research startup/company data and generate digests.
- `hyper-research` - General web research workflows.
- `openai-source-forge` - Source discovery and answer generation.

## Website To Product Or Agent

- `web-to-agent` - Analyze a website and scaffold typed Hyperbrowser tool functions.
- `flow-mapper` - Crawl a site and generate interaction/flow maps.
- `hyperbuild` - Generate app/workflow structures using Hyperbrowser nodes.
- `hyperdesign` - Analyze sites and produce design artifacts.
- `hyperpages` - Research and generate long-form content pages.

## Vertical Apps

- `universal-chatbot` - Chat with website content after scraping.
- `hb-ui-bot-app` - Screenshot and analyze UI/UX issues.
- `hb-pitchdeck` - Scrape a website and generate pitch deck content.
- `churnhunter` - CLI-style workflow for churn/opportunity discovery.
- `podcast-generator-ai`, `sora-research`, `mediresearch` - Domain-specific research/media examples.

Prefer using this index to select patterns, then inspect the upstream repo if full app context is needed.
