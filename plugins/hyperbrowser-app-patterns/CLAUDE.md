# Hyperbrowser App Patterns Plugin Instructions

Use this plugin for Hyperbrowser browser automation and scraping apps.

- Keep `HYPERBROWSER_API_KEY` server-side only.
- Validate URLs and request bodies before crawl/scrape/session calls.
- Bound crawl depth, page count, timeout, and output size.
- Clean up sessions reliably.
- Prefer structured API responses and progress reporting for long-running jobs.
