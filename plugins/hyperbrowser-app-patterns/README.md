# Hyperbrowser App Patterns Plugin

Curated agent guidance for building Hyperbrowser-based browser automation, scraping, crawling, research, screenshot analysis, competitor tracking, site-to-dataset, and web-to-agent applications.

This plugin packages patterns from `hyperbrowserai/hyperbrowser-app-examples` without copying the entire app catalog.

## Included Skill

- `hyperbrowser-app-patterns` - Build and review Hyperbrowser app workflows from curated examples and selected snippets.

## Compatibility

- Codex: `.codex-plugin/plugin.json` and `skills/hyperbrowser-app-patterns/SKILL.md`
- Claude: `.claude-plugin/plugin.json` and `skills/hyperbrowser-app-patterns/SKILL.md`
- GitHub Copilot: `.github/instructions/hyperbrowser-app-patterns.instructions.md`
