# Create Agent TUI Plugin

Scaffold customizable TypeScript terminal UIs for OpenRouter agents using `@openrouter/agent`.

This plugin packages the upstream OpenRouter `create-agent-tui` skill for the Spytrix AI marketplace.

## Included Skill

- `create-agent-tui` - Generate a TypeScript agent TUI with configurable tools, input styles, loaders, tool display modes, session persistence, and slash commands.

## Compatibility

- Codex: `.codex-plugin/plugin.json` and `skills/create-agent-tui/SKILL.md`
- Claude: `.claude-plugin/plugin.json` and `skills/create-agent-tui/SKILL.md`
- GitHub Copilot: repository-level `.github/copilot-instructions.md` plus `.github/instructions/create-agent-tui.instructions.md`
