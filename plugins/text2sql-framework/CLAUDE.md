# Text2SQL Framework Plugin Instructions

Use this plugin when building, reviewing, or operating Text-to-SQL agents.

- Prefer read-only database credentials.
- Require generated SQL to be executed before it is called verified.
- Capture traces for query attempts, errors, and final answers.
- Use `scenarios.md` to encode business rules and corrective examples.
- Keep scenario entries concise and scoped under `##` headings.
