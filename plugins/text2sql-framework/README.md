# Text2SQL Framework Plugin

Agent guidance for building and operating Text-to-SQL systems based on tool-driven schema exploration, verified SQL execution, traces, and `scenarios.md` feedback loops.

## Included Skill

- `text2sql-framework` - Build, integrate, review, and operate Text-to-SQL agents using the Text2SQL framework pattern.

## Compatibility

- Codex: `.codex-plugin/plugin.json` and `skills/text2sql-framework/SKILL.md`
- Claude: `.claude-plugin/plugin.json` and `skills/text2sql-framework/SKILL.md`
- GitHub Copilot: repository-level `.github/copilot-instructions.md` plus `.github/instructions/text2sql-framework.instructions.md`
