# Gas Town Agent Orchestration Plugin Instructions

Use this plugin for Gas Town multi-agent orchestration work.

- Use `gt prime` after compaction, restart, or context loss.
- Use `gt mail` for persistent inter-agent messages.
- Use `gt nudge` for immediate delivery to active agents.
- Use `bd` for issue tracking in Gas Town-managed repos.
- Use convoys to track batched work and `gt feed` for monitoring.
- Do not copy or vendor the full Gas Town runtime into this plugin.
